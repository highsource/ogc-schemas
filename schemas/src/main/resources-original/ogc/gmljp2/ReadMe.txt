OGC(r) GML in JPEG 2000 Encoding Standard (GMLJP2)
-----------------------------------------------------------------------

The OGC GML in JPEG 2000 Encoding Standard defines the means by which the OGC
Geography Markup Language (GML) Standard is used within JPEG 2000 images for
geographic imagery. The standard also provides packaging mechanisms for
including GML within JPEG 2000 data files and specific GML application schemas
to support the encoding of images within JPEG 2000 data files. JPEG 2000 is a
wavelet-based image compression standard that provides the ability to include
XML data for description of the image within the JPEG 2000 data file.

More information may be found at
 http://www.opengeospatial.org/standards/gmljp2

The most current schema are available at http://schemas.opengis.net/ 
and all OGC schemas may be downloaded in a complete bundle from
http://schemas.opengis.net/SCHEMAS_OPENGIS_NET.zip

The root (all-components) XML Schema Document, which includes
directly and indirectly all the XML Schema Documents, defined by
this Schema gmljp2.xsd.

* Latest version is: http://schemas.opengis.net/gmljp2/2.0/gmljp2.xsd *
