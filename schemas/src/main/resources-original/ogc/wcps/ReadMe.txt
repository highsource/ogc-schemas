OGC(r) WCS 1.1 Processing Service extension Standard 1.0.0
-------------------------------------------------------------------

The OGC Web Coverage Service Processing Extension is an extension of
the Web Coverage Service (WCS) Standard [OGC 07-067r5], based on the
Web Coverage Processing Service (WCPS) Language Interface Standard
[OGC 08-068r2]. This version of the extension applies to WCS version
1.1.2; with small changes, however, it is expected to also apply to
the next version of WCS.

The root (all-components) XML Schema Document, which includes
directly and indirectly all the XML Schema Documents, defined by
WCPS 1.0 is wcpsAll.xsd .

The WCPS 1.0.0 standard is defined in the following OGC documents:

 * 08-059r3 : Web Coverage Service (WCS) Processing Extension
   <http://portal.opengeospatial.org/files/?artifact_id=32416>

 * 08-068r2 : Web Coverage Processing Service (WCPS) Language Interface Standard
   <http://portal.opengeospatial.org/files/?artifact_id=32319>

 * 08-069r2 : Web Coverage Processing Service (WCPS) Language Interface Standard Abstract Test Suite
   <http://portal.opengeospatial.org/files/?artifact_id=32314>

 * 08-053r2 : Web Coverage Service (WCS) Processing Extension Abstract Test Suite
   <http://portal.opengeospatial.org/files/?artifact_id=32888>

More information may be found at 
 http://www.opengeospatial.org/standards/wcps
and 
 http://www.opengeospatial.org/standards/wcs
