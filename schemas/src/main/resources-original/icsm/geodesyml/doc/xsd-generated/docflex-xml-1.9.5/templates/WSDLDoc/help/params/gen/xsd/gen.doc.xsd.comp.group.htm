Specifies whether to generate the detailed Element Group Documentation for each global element group component.

Related Parameters:

This parameter provides the default value for the following parameters:
  • Details | Fragments | XSD Component Summary | Generate For | Element Groups
  • Navigation | Index Frame | All Components | XSD Components | Element Groups
  • Navigation | Index Frame | Namespace | XSD Components | Element Groups
  • Navigation | Index Frame | Schema | XSD Components | Element Groups
See Also:
"Details | XSD Component" parameter group, where you can specify what is included in the Element Group Documentation.
