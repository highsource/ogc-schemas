Specify whether to generate the Extensibility Elements section that shows the binding extensibility elements found in the WSDL operation input element.
Binding extensibility elements are non-WSDL elements that are used to bind the abstract WSDL definitions to a particular web-service implementation.
In the generated documentation, the extensibility elements will appear as a fragment of reproduced XML source, however, with only those special XML elements included in it.

Applies To:

Operations in WSDL binding
