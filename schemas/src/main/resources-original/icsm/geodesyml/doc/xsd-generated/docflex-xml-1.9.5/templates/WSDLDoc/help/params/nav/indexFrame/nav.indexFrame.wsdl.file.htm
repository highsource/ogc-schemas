Specify whether to generate WSDL File navigation indexes.

Such an index (a separate HTML page) is generated for every documented WSDL file. It may include:

  • The list of all XML schemas embedded in that WSDL file
  • The list of all top-level WSDL definitions found in that WSDL file
Nested Parameters:
Specify which of those lists are generated and what they include
Default Value:
Copied dynamically from the parameter: "Generate Details | WSDL File Overviews"
