With this parameter you can choose the destination of the component hyperlinks generated in this list.

Possible Choices:

"Component Details"

All component hyperlinks will go to the component details, which are the Component Documentation pages generated for each component.

The generation of the Component Documentation is controlled by:

  • The parameters in the "Generate Details" parameter group corresponding to particular component types
  • "Details | Component Documentation" parameter group
"XML Source"
In this case, each component hyperlink will be directed to the definition of this component within the reproduced full XML schema source.

That effectively will allow you to use the component list as an index to your XML schema source.

Note that the full XML schema source is reproduced on the Schema Overview page, which is controlled by the "Details | Schema Overview | XML Source" parameter group.

Notes:
  • When a particular hyperlink destination corresponding to this parameter setting does not exist (has not been generated), an alternative destination will be tried (e.g. the component's XML source instead of the Component Documentation and vice-versa).
  • All pages loaded by the component hyperlinks will be targeted to the Detail Frame (see also "Navigation | Detail Frame" parameter group).
