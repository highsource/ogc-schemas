This group of parameters controls the detailed content of WSDL Definition Documentation pages/blocks that can be generated for the following WSDL definitions:
  • Services
  • Bindings
  • Port Types
  • Messages
Certain parameters in this group apply to all WSDL definitions. Others are specific to only some definition types.

See Also:

The corresponding parameters in the "Generate Details | WSDL Definitions" parameter group:
  • "Services"
  • "Bindings"
  • "Port Types"
  • "Messages"
With these parameters, you can select which of those definitions must be documented.
