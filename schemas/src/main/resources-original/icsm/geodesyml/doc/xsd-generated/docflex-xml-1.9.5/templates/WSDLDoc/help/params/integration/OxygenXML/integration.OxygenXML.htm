Enables the OxygenXML Integration.

When this parameter is unselected (false), OxygenXML won't be called.

Note: The parameter is enabled only when the OxygenXML Integration has been setup. For more details, please see the integration README found in your DocFlex/XML installation: {docflex-xml}/integrations/OxygenXML/README.html
