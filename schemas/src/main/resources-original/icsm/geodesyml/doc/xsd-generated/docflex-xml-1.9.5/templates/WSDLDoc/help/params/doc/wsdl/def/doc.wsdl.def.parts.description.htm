Specify whether to include the part description.

The description text is obtained from the <wsdl:documentation> element found by the following path:

wsdl:operation/wsdl:documentation
However, when that description is absent, an alternative description may be used, which is searched for according to the settings in the parameter group: "Details | WSDL Definition | Alternative Descriptions | Message Part".

See Also:

"Processing | Descriptions" parameter group, where you can specify how descriptions are processed and displayed.
