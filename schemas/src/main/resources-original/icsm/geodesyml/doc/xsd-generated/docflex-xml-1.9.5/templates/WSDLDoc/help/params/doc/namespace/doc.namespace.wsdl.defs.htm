Specify whether to generate the summary section of the WSDL definitins that belong to the given namespace.

Default Value:

Copied dynamically from the value of the parameter: "Generate Details | WSDL Definitions".
Nested Parameters:
Specify the types of the WSDL definitions to be shown in the summary tables, the ordering of the definitions and what information must be displayed about each of them.

The default values of all these parameters are copied from the values of the corresponding parameters in the group: "Details | Fragments | WSDL Definition Summary"
