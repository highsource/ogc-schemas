Specifies whether to generate the detailed Message Documentation for each WSDL message definition.

Related Parameters:

This parameter provides the default value for the following parameters:
  • Details | Fragments | WSDL Definition Summary | Generate For | Messages
  • Navigation | Index Frame | All Components | WSDL Definitions | Messages
  • Navigation | Index Frame | Namespace | WSDL Definitions | Messages
  • Navigation | Index Frame | WSDL File | WSDL Definitions | Messages
See Also:
"Details | WSDL Definition" parameter group, where you can adjust the exact content of the Message Documentation.
