Specifies whether to generate the Overview Summary page/block.

The Overview Summary may be generated only once for the entire WSDL/XSD documentation. It includes:

  • Summary of namespaces.
  • Summary of all WSDL files and XML schemas documented.
It appears the first in the details frame of framed HTML documentation (when it is started) or on the first page of single file documentation (RTF).

The exact content of the Overview Summary page/block is specified with the parameter group:

"Details | Overview Summary"
