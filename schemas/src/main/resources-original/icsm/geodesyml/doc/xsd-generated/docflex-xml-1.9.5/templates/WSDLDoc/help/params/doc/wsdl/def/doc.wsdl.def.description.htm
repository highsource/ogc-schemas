Specify whether to generate the "Description" section of the WSDL Definition Documentation page/block.

The description text is obtained from the <wsdl:documentation> element found by the following path:

wsdl:definition/wsdl:documentation
where 'wsdl:definition' is the particular XML element of the WSDL definition (e.g. <wsdl:portType>).

Applies To:

All definitions
See Also:
"Processing | Descriptions" parameter group, where you can specify how descriptions are processed and displayed.
