This parameter allows you to control the sorting of WSDL files throughout the documentation.

It directly affects:

  1. The ordering of WSDL files in the Overview Frame (see also parameter: "Navigation | Overview Frame").
  2. In the case of framed documentation (generated by FramedDoc.tpl main template), the order, in which WSDL files become fully processed and documented (this includes the generation of WSDL File Overview and WSDL Definition Documentation pages for a WSDL file). It determines which WSDL file gets documented the first, and that, in turn, may affect the initial content of the frames:
    • Index Frame. (See parameter: "Navigation | Index Frame")
    • Detail Frame. (See parameter: "Navigation | Detail Frame")
  3. In the case of single file documentation (generated by PlainDoc.tpl main template), the order, in which the WSDL File Overview blocks (generated for each WSDL file) follow each other.
The parameter has the following meaning:
  • When it is selected (true), WSDL files are sorted alphabetically by their names (ignoring letter case).
  • When the parameter is unselected (false), WSDL files will follow in a certain "physical" order determined by what files have been specified initially and how they import other WSDL files.
