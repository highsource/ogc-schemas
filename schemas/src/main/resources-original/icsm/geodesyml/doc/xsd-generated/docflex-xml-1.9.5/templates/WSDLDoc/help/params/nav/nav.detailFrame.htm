This parameter allows you to customize the initial content of the Detail Frame, which occupies the largest (right) part of the frameset:
(All the documentation informative content appears in this frame. Other frames are used mainly for navigation purpose.)

Possible Choices:

"Overview Summary"

Preload the Overview Summary page.

The generation of this page is controlled by the parameters/groups:

  • Generate Details | Overview Summary
  • Details | Schema Overview
"All Component Summary"
Preload the All Component Summary page.

The generation of this page is controlled by the parameters/groups:

  • Generate Details | All Component Summary
  • Details | All Component Summary
"Namespace Overview"
Preload one of the Namespace Overview pages.

As there may be several of such pages (a separate page for each namespace), the preloaded will be one, which has been generated the first. That is controlled by the parameter: "Generate Details | Sorting | Namespaces".

The generation of Namespace Overview pages is controlled by the parameters/groups:

  • Generate Details | Namespace Overview
  • Details | Namespace Overview
"WSDL File Overview"
Preload one of the WSDL File Overview pages.

As there may be several of such pages (a separate page for each WSDL file), the preloaded will be one, which has been generated the first. That is controlled by the parameter: "Generate Details | Sorting | WSDL Files".

The generation of WSDL File Overview pages is controlled by the parameters/groups:

  • Generate Details | Namespace Overview
  • Details | Namespace Overview
"Schema Overview"
Preload one of the Schema Overview pages.

As there may be several of such pages (a separate page for each XML schema), the preloaded will be one, which has been generated the first.

Here is how it is determined: The first get processed the XML schemas embedded in WSDL files, in the same order as the WSDL files themselves. That is controlled by the parameter: "Generate Details | Sorting | WSDL File". Then, XSD files are processed in the order controlled by the parameter: "Generate Details | Sorting | Schemas".

The generation of Schema Overview pages is controlled by the parameters/groups:

  • Generate Details | Schema Overview
  • Details | Schema Overview
Note:
When the specified index page is not available, what is loaded initially will be chosen according to the following priority list:
  1. Overview Summary
  2. All Component Summary
  3. Namespace Overview
  4. WSDL File Overview
  5. Schema Overview
  6. WSDL Definition Documentation
  7. XSD Component Documentation
  8. Namespace Bindings Report
