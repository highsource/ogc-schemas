Specifies whether to generate the detailed Global Attribute Documentation for each global attribute component.

Related Parameters:

This parameter provides the default value for the following parameters:
  • Details | Fragments | XSD Component Summary | Generate For | Global Attributes
  • Navigation | Index Frame | All Components | XSD Components | Global Attributes
  • Navigation | Index Frame | Namespace | XSD Components | Global Attributes
  • Navigation | Index Frame | Schema | XSD Components | Global Attributes
See Also:
"Details | XSD Component" parameter group, where you can specify what is included in the Global Attribute Documentation.
