Specifies whether to generate details of XSD components (that is XSD Component Documentation pages/blocks).

The exact content of the XSD Component Documentation is controlled by the parameter group:

"Details | XSD Component"
Nested Parameters:
Specify particular types of XSD component, for which the details block will be generated.
Related Parameters:
This parameter provides the default value for the following parameters:
  • Details | All Component Summary | XSD Component Summary
  • Details | Namespace Overview | XSD Component Summary
  • Details | Schema Overview | XSD Component Summary
  • Navigation | Index Frame | All Components | XSD Components
  • Navigation | Index Frame | Namespace | XSD Components
  • Navigation | Index Frame | Schema | XSD Components
