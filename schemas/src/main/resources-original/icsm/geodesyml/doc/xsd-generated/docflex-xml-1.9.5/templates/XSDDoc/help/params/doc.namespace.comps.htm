Specify whether to generate summary tables of global components and local elements that belong to the given namespace (that is defined in all XML schemas that target this namespace).

Nested Parameter Group:

Specify the types of the components to be shown in the summary tables, the ordering of the components and which information must be displayed about each component.

The default values of the parameters selecting the components are calculated dynamically from the corresponding parameters of the "Generate Details" group. That is, by default, only those components will appear in the summary tables which you have already specified to be documented in full. The parameters just allow you to override this.
