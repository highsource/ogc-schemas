Specifies whether to generate the WSDL File Overview pages/blocks.

When specified, the WSDL File Overview will be generated for each WSDL file being documented. It includes:

  • General information about this WSDL file (profile)
  • Description
  • Summary of XML schemas embedded in this WSDL file
  • Summary of all WSDL definitions found in this WSDL file
The exact content of the WSDL File Overview pages/blocks is controlled by the parameter group:
"Details | WSDL File Overview"
Related Parameters:
This parameter provides the default value for the following parameters:
  • Navigation | Index Frame | WSDL File
  • Navigation | Index Frame | Namespace | WSDL Files
