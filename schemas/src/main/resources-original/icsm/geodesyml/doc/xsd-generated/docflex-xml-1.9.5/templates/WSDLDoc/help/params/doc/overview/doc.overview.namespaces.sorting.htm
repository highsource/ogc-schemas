Specify how the Namespace Summary is sorted.

The parameter has the following meaning:

  • When it is selected (true), the namespaces will be sorted alphabetically by their URIs (ignoring letter case).
  • When the parameter is unselected (false), the namespaces won't be sorted anyhow. The ordering of namespaces will be determined by what WSDL/XSD files have been specified initially and how they reference (e.g. import) other WSDL/XSD files.
Default Value:
Copied dynamically from the parameter: "Generate Details | Sorting | Namespaces"
