Specifies whether to generate the detailed Simple Type Documentation for each global simple type component.

Related Parameters:

This parameter provides the default value for the following parameters:
  • Details | Fragments | XSD Component Summary | Generate For | Simple Types
  • Navigation | Index Frame | All Components | XSD Components | Simple Types
  • Navigation | Index Frame | Namespace | XSD Components | Simple Types
  • Navigation | Index Frame | Schema | XSD Components | Simple Types
See Also:
"Details | XSD Component" parameter group, which controls the exact content of the Simple Type Documentation.
