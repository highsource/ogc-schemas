Specify whether to generate Operation Summary section that lists all operations (or operation bindings) defined in the given WSDL portType or binding.

The summary information about each operation includes its name and description (the first sentence). The description text is supposed to come from the <wsdl:documentation> element found by the path:

wsdl:operation/wsdl:documentation
However, when that description is absent, an alternative description may be used, which is searched for according to the settings in the parameter group: "Details | WSDL Definition | Alternative Descriptions | Operation".

Applies To:

  • Port Types
  • Bindings
Nested Parameters:
Select precisely for which top WSDL definitions (portTypes and bindings) the operation summary is generated
See Also Parameters:
  • Details | WSDL Definition | Alternative Descriptions | Operation
  • Details | WSDL Definition | Definition Profile | Includes
  • Details | WSDL Definition | Operation Detail
