Specifies whether to generate the Schema Overview pages/blocks.

When specified, the Schema Overview will be generated for each XML schema being documented. It includes:

  • General information about this XML schema (profile)
  • Description (collected by all annotations of <xs:schema> element)
  • Summary of all XSD components found in this schema
The exact content of the Schema Overview pages/blocks is controlled by the parameter group:
"Details | Schema Overview"
Related Parameters:
This parameter provides the default value for the following parameters:
  • Navigation | Index Frame | Schema
  • Navigation | Index Frame | Namespace | Schemas
