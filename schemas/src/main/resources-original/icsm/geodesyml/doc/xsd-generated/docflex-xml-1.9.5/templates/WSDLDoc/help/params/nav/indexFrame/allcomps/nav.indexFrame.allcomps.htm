Specify whether to generate the index of all components.

Only one such an index (HTML file) is generated. It may include:

  • The list of all top-level WSDL definitions (services, bindings, portTypes, messages) defined throughout all documented WSDL files.
  • The list of all global XSD components and local elements defined throughout all documented XML schemas.
Default Value:
Copied dynamically from the value of the parameter: "Generate Details | All Component Summary".
Nested Parameters:
Specify which of those lists are generated and what they include
