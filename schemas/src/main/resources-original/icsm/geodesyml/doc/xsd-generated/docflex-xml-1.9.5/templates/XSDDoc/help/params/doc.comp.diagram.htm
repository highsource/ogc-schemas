Specify whether to generate/include the graphic diagram representation of the XSD component.

Currently, the diagrams can be generated only by an extension of DocFlex/XML that employs the functionality of other software. These are the integrations with:

  • Altova XMLSpy (see "Integrations | XMLSpy" parameter group)
  • Oxygen XML (see "Integrations | Oxygen XML" parameter group)
Applies To:
  • Elements
  • Complex Types
  • Element Groups
Nested Parameters:
Control how to direct hyperlinks from the diagram image.
