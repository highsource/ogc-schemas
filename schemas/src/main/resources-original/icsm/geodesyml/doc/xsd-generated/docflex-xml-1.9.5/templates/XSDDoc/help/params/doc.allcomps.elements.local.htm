Specifies whether to include local elements in the Element Summary.

(Local elements are the locally defined element components. For more information, see "About Local Elements" below.)

Possible Choices:

"all"

Include all local elements.
"with complex type only"
Include only local elements with complex types.
"none"
Do not include local elements.

Default Value:

The default value of this parameter is copied dynamically from the value of the "Generate Details | Elements | Local Elements" parameter.
