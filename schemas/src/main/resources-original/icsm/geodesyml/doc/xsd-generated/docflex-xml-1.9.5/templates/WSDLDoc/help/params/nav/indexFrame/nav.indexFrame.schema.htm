Specify whether to generate XML schema navigation indexes.

Such an index (a separate HTML page) is generated for every documented XML schema (both XSD file and XML schema embedded in WSDL). It may include:

  • The list of all global components defined in that schema
  • The list of all local elements defined in that schema
Default Value:
Copied dynamically from the parameter: "Generate Details | Schema Overviews"
Nested Parameters:
Specify which of those lists are generated and what they include
