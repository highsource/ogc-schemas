Specifies whether to generate the detailed Port Type Documentation for each WSDL portType definition.

Related Parameters:

This parameter provides the default value for the following parameters:
  • Details | Fragments | WSDL Definition Summary | Generate For | Port Types
  • Navigation | Index Frame | All Components | WSDL Definitions | Port Types
  • Navigation | Index Frame | Namespace | WSDL Definitions | Port Types
  • Navigation | Index Frame | WSDL File | WSDL Definitions | Port Types
See Also:
"Details | WSDL Definition" parameter group, where you can adjust the exact content of the Port Type Documentation.
