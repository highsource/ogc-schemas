Specify whether to include in the summary item the description of the WSDL definition and which part of it.

The full description text is obtained from the <wsdl:documentation> element found by the following path:

wsdl:definition/wsdl:documentation
where 'wsdl:definition' is the particular XML element of the WSDL definition (e.g. <wsdl:portType>).

Possible Choices:

"first sentence"

Include only the first sentence of the description text.
"full"
Include the full description.
"none"
Do not include the description.

See Also:

"Processing | Descriptions" parameter group, where you can specify how descriptions are processed and displayed.
