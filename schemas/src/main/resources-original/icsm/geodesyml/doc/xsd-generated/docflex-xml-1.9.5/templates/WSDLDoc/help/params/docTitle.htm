Specify the overall title for the whole documentation.

In particular:

  1. This will be the main part of the documentation window title (i.e. the title of the HTML browser window that displays the documentation).
  2. In the case of RTF output format, the text specified with this parameter will appear on page footers.
  3. This parameter provides also the default value for "Details | Overview Summary | Title" parameter, which specifies the title of Overview Summary page/block.
  4. Equally, it provides the default value for "Navigation | Overview Frame | Title" parameter, which specifies the title of Overview Frame.
