Specify whether to generate the Schema File Summary table that enumerates all documented XSD files along with a few details about each of them.

Note:

This summary does not include the XML schemas embedded in WSDL files. Those are mentioned in the summary items of the respective WSDL files themselves. See also parameter: "Details | Overview Summary | WSDL File Summary | WSDL File Profile | Types / Schemas"
Nested Parameters:
Specify how the schema summary is sorted and which details are shown about each XML schema.

The default values of all these parameters are copied from the values of the corresponding parameters in the group: "Details | Fragments | Schema Summary"
