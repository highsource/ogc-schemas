Specifies whether to remove all <wsdl:documentation> elements from the reproduced XML source of the whole WSDL file.

You may want to exclude the <wsdl:documentation> elements from the reproduced XML source because such elements may occupy a lot of space (especially, when you use XHTML to format your descriptions), so they could overwhelm anything else making it difficult to read other important things. Moreover, the actual content of the <wsdl:documentation> elements may be already shown (as a formatted text) in the corresponding Description sections of the documentation.
