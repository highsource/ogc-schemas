Specify whether to document XML schemas embedded within WSDL files (i.e. those found within <wsdl:definitions>/<wsdl:types> elements).

For example, if you document only WSDL files and want to see only WSDL definitions in your documentation, you should unselect these parameters:

  1. Generate Details | For | WSDL/XSD Files | Referenced | XSD
  2. Generate Details | For | Embedded Schemas
