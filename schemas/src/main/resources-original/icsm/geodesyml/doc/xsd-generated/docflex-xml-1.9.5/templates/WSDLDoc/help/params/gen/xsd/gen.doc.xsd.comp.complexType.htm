Specifies whether to generate the detailed Complex Type Documentation for each global complex type component.

Related Parameters:

This parameter provides the default value for the following parameters:
  • Details | Fragments | XSD Component Summary | Generate For | Complex Types
  • Navigation | Index Frame | All Components | XSD Components | Complex Types
  • Navigation | Index Frame | Namespace | XSD Components | Complex Types
  • Navigation | Index Frame | Schema | XSD Components | Complex Types
See Also:
"Details | XSD Component" parameter group, where you can specify what will be included in the Complex Type Documentation.
