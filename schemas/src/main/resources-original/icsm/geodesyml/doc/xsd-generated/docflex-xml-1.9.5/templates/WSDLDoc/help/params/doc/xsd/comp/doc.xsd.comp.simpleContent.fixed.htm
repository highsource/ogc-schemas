Specify whether to show the fixed value assigned to the given simple content.

This applies to global element and global attribute components. The default value is specified using 'fixed' attribute of the XSD element defining the component.

In the case of local elements, this item may be not applicable. When an element is declared locally in several places, its fixed value may be different at each location, therefore, it is senseless to display here just one of those values. (For more details about documenting of local elements, please see the description of the parameter: "Generate Details | XSD Components | Elements | Local Elements".)

Applies To:

  • Elements
  • Global Attributes
