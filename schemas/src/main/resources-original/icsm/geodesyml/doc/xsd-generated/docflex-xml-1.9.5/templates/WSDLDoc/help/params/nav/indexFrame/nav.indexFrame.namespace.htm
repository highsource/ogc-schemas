Specify whether to generate namespace navigation indexes.

Such an index (a separate HTML page) is generated for every documented namespace. It may include:

  • The list of all WSDL files targeting that namespace.
  • The list of all XML schemas targeting that namespace (both XSD files and XML schemas embedded in WSDL).
  • The list of all top-level WSDL definitions, which belong to that namespace.
  • The list of all global XSD components and local elements, which belong to that namespace.
Default Value:
Copied dynamically from the parameter: "Generate Details | Namespace Overviews"
Nested Parameters:
Specify which of those lists are generated and what they include
