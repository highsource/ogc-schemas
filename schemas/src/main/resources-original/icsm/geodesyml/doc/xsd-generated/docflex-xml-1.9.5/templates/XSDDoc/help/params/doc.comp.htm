This group of parameters controls the detailed content of the Component Documentation pages/blocks that can be generated for all XML schema global components and all local element components. In particular, this applies to the following types of components:
  • Elements (both global and local ones)
  • Complex Types
  • Simple Types
  • Element Groups
  • Global Attributes
  • Attribute Groups
Certain parameters in this group apply to all component types. Others are specific to only some component types.

See Also:

The corresponding parameters in "Generate Details" parameter group:
  • "Elements"
  • "Local Elements"
  • "Complex Types"
  • "Simple Types"
  • "Element Groups"
  • "Global Attributes"
  • "Attribute Groups"
With these parameters, you can select which of those components should be documented.
