Specifies whether to generate the All Component Summary page/block.

This page/block may be generated only once for the entire WSDL/XSD documentation. It contains summaries of all WSDL definitions and XSD components of the specified types throughout all WSDL/XSD files documented.

The exact content of the All Component Summary page/block is controlled by the parameter group:

"Details | All Component Summary"
Related Parameters:
This parameter provides the default value for the following parameter:
  • Navigation | Index Frame | All Components
