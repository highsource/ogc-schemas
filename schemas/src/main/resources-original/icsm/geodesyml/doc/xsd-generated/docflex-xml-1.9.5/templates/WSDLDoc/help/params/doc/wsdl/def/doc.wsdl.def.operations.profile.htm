Specify whether to generate the Operation Profile section, which contains the summary information about the operation, including:
  • Operation Type (e.g. request-response, one-way, solicit-response, notification)
  • The link to the reproduced XML source of the operation (when available)
  • For portType operation: in which bindings it is used
  • For operation binding: by which portType operation it is specified

Applies To:

All operations (portType/binding)
