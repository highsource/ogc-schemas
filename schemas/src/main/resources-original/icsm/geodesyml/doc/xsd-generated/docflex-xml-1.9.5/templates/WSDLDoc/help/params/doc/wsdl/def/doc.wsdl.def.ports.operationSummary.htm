Specify whether to include the Operation Summary section that lists all operations provided by the port.

That operation summary replicates a similar summary in Binding Documentation (see parameter: "Details | WSDL Definition | Operation Summary").

The summary information about each operation includes its name and description (the first sentence). The description text comes from the <wsdl:documentation> element found by the path:

wsdl:binding/wsdl:operation/wsdl:documentation
However, when that description is absent, an alternative description may be used, which is searched for according to the settings in the parameter group: "Details | WSDL Definition | Alternative Descriptions | Operation".
