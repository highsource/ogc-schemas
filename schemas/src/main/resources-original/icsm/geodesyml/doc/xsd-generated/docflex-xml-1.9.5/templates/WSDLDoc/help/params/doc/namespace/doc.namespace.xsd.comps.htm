Specify whether to generate the summary section of the XSD components that belong to the given namespace.

Default Value:

Copied dynamically from the value of the parameter: "Generate Details | XSD Components".
Nested Parameters:
Specify the types of the components to be shown in the summary tables, the ordering of the components and which information must be displayed about each component.

The default values of all these parameters are copied from the values of the corresponding parameters in the group: "Details | Fragments | XSD Component Summary"
