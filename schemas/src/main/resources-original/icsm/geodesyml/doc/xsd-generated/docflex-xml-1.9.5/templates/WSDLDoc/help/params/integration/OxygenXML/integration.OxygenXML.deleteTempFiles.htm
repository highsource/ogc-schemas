Specify whether to delete all files temporarily generated by OxygenXML (as well as the integration files required for it).

When this parameter is selected (true), the entire subdirectory:

{documentation_files}/OxygenXML
will be deleted on the finishing of WSDLDoc.

When this parameter is unselected (false), all original output files generated by OxygenXML will be left. (Typically, this is needed to investigate some problems about the integration.)
