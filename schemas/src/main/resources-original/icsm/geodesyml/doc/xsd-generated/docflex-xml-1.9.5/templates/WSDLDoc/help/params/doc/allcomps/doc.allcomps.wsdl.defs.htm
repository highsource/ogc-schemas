Specify whether to generate summary section of all WSDL definitions of the specified types (see nested parameters) throughout all WSDL files being documented.

Default Value:

Copied dynamically from the value of the parameter: "Generate Details | WSDL Definitions".
Nested Parameters:
Specify the types of the WSDL definitions to be shown in the summary tables, the ordering of the definitions and what information must be displayed about each of them.

The default values of most of these parameters are copied from the values of the corresponding parameters in the group: "Details | Fragments | WSDL Definition Summary"
