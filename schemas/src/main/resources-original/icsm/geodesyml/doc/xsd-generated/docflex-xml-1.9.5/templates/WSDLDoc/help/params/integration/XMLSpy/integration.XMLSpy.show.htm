Specify whether XMLSpy application (IDE) should show up, when it is called from DocFlex to process the XML schemas.

When "Integrations | XMLSpy | Quit XMLSpy" parameter was unselected (false), the XMLSpy IDE will remain visible even after finishing DocFlex/XML. So, you can use it and quit by standard means.

See Also Parameter:

Integrations | XMLSpy | Quit XMLSpy
