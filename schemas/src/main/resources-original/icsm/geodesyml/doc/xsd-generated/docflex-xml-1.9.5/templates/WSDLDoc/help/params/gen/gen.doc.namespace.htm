Specifies whether to generate the Namespace Overview pages/blocks.

When selected, the Namespace Overview block is generated for each namespace being documented (i.e. targeted by at least one of the documented WSDL files and XML schemas). It includes:

  • Summaries of all WSDL files and XML schemas targeting that namespace.
  • Summaries of all WSDL definitions and XSD components defined in that namespace.
The exact content of the Namespace Overview page/block is controlled by the parameter group:
"Details | Namespace Overview"
Related Parameters:
This parameter provides the default value for the following parameter:
  • Navigation | Index Frame | Namespace
