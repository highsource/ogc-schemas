This parameter allows you to specify directly the names of the elements to be documented in separate Element Documentation blocks.

The elements should be specified with their local names (without namespace prefixes) separated with newlines or semicolons (';'). Only the element with the specified names will be documented.

When an element name is shared by several element components, including local ones, all of them are to be documented (according to the setting of Local Elements parameter).

When the list of names is empty (the parameter is not specified), all elements will be documented (considering other filtering criteria).
