Enables the XMLSpy Integration. (See How XMLSpy Integration Works below.)

When this parameter is unselected (false), XMLSpy won't be called.

Note: The parameter is enabled only when the XMLSpy Integration has been setup. For more details, please see the integration README found in your DocFlex/XML installation: {docflex-xml}/integrations/XMLSpy/README.html
